A Guide to Modding in Gorilla Tag

Disclaimer

All modifications (mods) for Gorilla Tag are considered bannable by the game’s developers. Use of mods is done entirely at your own risk, and may result in account restrictions or permanent bans.

⸻

Requirements

Before getting started, ensure you have the following:

* A compatible PC
* Gorilla Tag installed via Steam
* Access to the official Gorilla Tag Discord server (link to be provided)

⸻

Installation Process

1. Download Monke Mod Manager
    Obtain Monke Mod Manager from the official Gorilla Tag Discord server.
2. Install or Update Mods
    * Open Monke Mod Manager
    * Click “Install/Update” to set up the required files
3. Access the Mods Folder
    * Navigate to the Mods tab
    * Select “Open Mods Folder”
4. Add Mod Files
    * Download the desired .dll mod files (e.g., GShirts or other custom mods)
    * Drag and drop the .dll files into the Plugins folder
5. Launch the Game
    * Start Gorilla Tag through Steam
    * Use the assigned keybind to activate or open your mod menu, depending on the mod’s functionality

⸻

Creating Your Own Mods

Users may also create their own mods by:

* Using pre-made templates
* Developing from scratch with appropriate tools and programming knowledge

⸻

Why CosmeticX Is Bannable

CosmeticX is prohibited because it involves the unauthorized use of in-game content. Specifically, it allows users to access paid cosmetics without purchasing them.

This falls under piracy, which refers to obtaining or using paid digital content without proper authorization. Even if these cosmetics are client-sided (visible only to the user), their use still violates the game’s terms of service.

As a result, using CosmeticX can lead to disciplinary action, including bans.
